# Open_A-Life
![Open A-Life Logo](git/logo.svg)
## About
A modular and open source version of the A-Life system found in the S.T.A.L.K.E.R. Designed to be easily embedded into any application!
Open A-Life simulates a world in the form of a 2D grid of cells. This world is then populated by different factions which may interact/fight with eachother in various different ways. These could be traders seeking deals, diseases ravaging towns, armies seeking more land, or whatever you may want to add.
